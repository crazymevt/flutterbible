# Content Manager

The Content Manager is your gateway to expanding the StudyBible app. It draws on several large, free libraries of biblical resources so you can build the exact study library you want.

> 🌐 **Needs internet** — Browsing the catalogs and downloading modules both require a connection. Once a module is downloaded it works completely offline.

## Catalogs
The Content Manager has tabs for each source:

- **Installed** — everything currently on your device, with the option to remove what you no longer need.
- **ph4.org Catalog** — a broad collection of Bibles, commentaries, dictionaries, and devotionals.
- **OSIS Catalog** — translations organized by language, so you can find Scripture in the language you need.
- **CrossWire Catalog** — the classic, massive CrossWire/SWORD library.

## Downloading Modules
1. Open the **Content Manager** from the hamburger menu (App Drawer).
2. Pick a catalog tab and browse or use the search box to filter it.
3. You'll find **Bibles**, **Commentaries**, **Dictionaries**, and **Devotionals**.
4. Tap the download icon next to any module to install it.
5. Once downloaded, the module is available completely offline!

## Deleting Modules
To free up space, remove a module from the **Installed** tab using its trash icon.

## Module Formats
StudyBible supports standard module formats, including the CrossWire/SWORD formats (zText, zCom, RawText). We continually expand support for more advanced parsing features, such as Greek/Hebrew morphology modules.
