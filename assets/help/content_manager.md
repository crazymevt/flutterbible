# Content Manager

The Content Manager is your gateway to expanding the StudyBible app. It connects to the massive, free CrossWire library of biblical resources.

> 🌐 **Needs internet** — Browsing the catalog and downloading modules both require a connection. Once a module is downloaded it works completely offline.

## Downloading Modules
1. Open the **Content Manager** from the hamburger menu (App Drawer).
2. Browse the categories: **Bibles**, **Commentaries**, and **Dictionaries**.
3. Tap the download icon next to any module to install it on your device.
4. Once downloaded, the module is available completely offline!

## Deleting Modules
To free up space on your device, you can delete a module by swiping left on it (or tapping the trash icon) in the Content Manager.

## Module Formats
StudyBible supports standard CrossWire module formats (zText, zCom, RawText). We continually update the app to support more advanced parsing features for Greek/Hebrew morphology modules!
