# Search

The global search allows you to search across all downloaded Bibles, your personal journals, sermons, and notes simultaneously. It is blazing fast and runs entirely locally.

## Tips for Searching
- **Exact Match:** By default, search looks for exact phrase matches.
- **Proximity Search (NEAR):** Use `~` followed by a number to find words near each other. For example, `faith ~10 works` finds verses where "faith" and "works" are within 10 words of each other, in any order. You can chain this: `Jesus ~5 wept ~5 bitterly`.
- **Prefix Match:** You can use an asterisk `*` to perform prefix searches (e.g., `bapt*` will find baptize, baptism, baptist).
- **Navigation (Goto):** You can enter a book reference directly into the search bar to jump straight there! For example, typing "John 3:16" or "1 Cor 13" will instantly take you to that chapter in the Reader.

## Scope Filtering
You can restrict your search to the Old Testament, New Testament, or a specific book using prefixes:
- `ot: love` - Searches only in the Old Testament.
- `nt: love` - Searches only in the New Testament.
- `John: light` - Searches only in the book of John.

## Search Index
If you ever notice that search results look strange or contain formatting markup, you can rebuild the search index manually from the Settings menu.
